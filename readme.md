For this script to work, you need to start by installing Nu and git

```
sudo pacman -S nushell git
```

Then clone the repo 

```
git clone https://github.com/IndigoCurnick/dotfiles.git
```

Run the script 

```
sudo bash dotfiles/setup.sh
```
